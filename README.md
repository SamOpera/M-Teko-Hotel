# 🏨 M-Teko Hotel Website

This is the official website for **M-Teko Hotel**, a hospitality business based in Lagos, Nigeria.  
The project was developed as part of my role as a **Web Developer** for M-Teko Hotel from **Feb 2025 to April 2025**.

## 🌐 Live Site  
Visit: [www.mtekohotel.com](http://www.mtekohotel.com)

---

## 🛠️ Features

- ✅ Responsive homepage with hotel information and visual branding
- ✅ Image gallery showcasing room types and amenities
- ✅ Contact form integrated with hotel management email
- ✅ Room booking form with dropdown and calendar selection
- ✅ Footer with links, social media, and map integration

---

## 💻 Technologies Used

- **HTML5** – Semantic structure and layout  
- **CSS3** – Styling, animations, responsive design  
- **JavaScript** – Basic interactivity and form validation  
- **Bootstrap** – Grid system and layout responsiveness  
- **Git** – Version control
